# api

## Build Setup

``` bash
# Install dependencies
npm install

# Start developing with REPL
npm run dev

# Start production
npm start

# Run unit tests
npm test

# Run continuous test mode
npm run ci

# Run TSLint
npm run lint
```

## Run in Docker

**Build Docker image**
```bash
$ docker build -t api .
```

**Start container**
```bash
$ docker run -d api
```
